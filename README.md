## Projects

###  1. Diabetes Readmission Prediction
**Tools:** R, Machine Learning  

Developed predictive models using over 100,000 patient records to estimate 30-day hospital readmission risk. Implemented multiple machine learning algorithms including Ridge, LASSO, Random Forest, LightGBM, and XGBoost.

**Key Results:**
- Best model: XGBoost (AUC = 0.68)  
- Addressed class imbalance  
- Identified key predictors (hospital stay duration, prior admissions)  

---

###  2. Marketing Response Prediction & Customer Segmentation
**Tools:** Python  

Built supervised learning models to predict customer responses to marketing campaigns. Performed data preprocessing, feature engineering, and model evaluation.

**Key Results:**
- Achieved AUC ≈ 0.70  
- Developed customer segmentation strategies  
- Improved targeting efficiency by 15–20%  

---

###  3. California Temperature Forecasting (Time Series Analysis)
**Tools:** R, SARIMA  

Analyzed monthly temperature data (1895–2023) and built SARIMA models to generate forecasts.

 **Project Link:**  
https://github.com/ankwasadoreen/california-temperature-forecasting-sarima  

**Key Results:**
- Generated 36-month forecasts  
- RMSE: 0.6–0.8°C  
- Identified seasonal patterns and warming trends
